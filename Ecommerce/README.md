
# 🛒 Amazon E-Commerce Website Clone

## 📌 Project Overview
This project is a front-end clone of the Amazon e-commerce website.  
It replicates the design and basic user interface of Amazon using HTML, CSS, and JavaScript.

The goal of this project is to improve my front-end development skills by building a real-world website layout.

---

## 🚀 Features

- ✅ Responsive Navigation Bar
- ✅ Product Listing Section
- ✅ Search Bar UI
- ✅ Add to Cart Button (UI Based)
- ✅ Hero Section Banner
- ✅ Footer Section
- ✅ Clean and Modern Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
